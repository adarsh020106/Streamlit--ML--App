**# 🚖 Ride Demand Prediction System**



**## 📌 Description**



This project is a Machine Learning-based web application that predicts the \*\*number of weekly riders\*\* based on various economic and demographic factors.



The application is built using **\*\*Streamlit\*\*** for an interactive user interface and a trained ML model for prediction.



\---



**## 🚀 Features**

\- Predicts weekly ride demand using Machine Learning

\- Simple and user-friendly UI

\- Real-time prediction based on user input

\- Lightweight and fast execution



\---



**## 📊 Input Parameters**

The model takes the following inputs:

\- 💰 Price per week

\- 👥 Population

\- 💵 Monthly income

\- 🅿️ Average parking per month



\---



**## 📈 Output**

\- 🎯 Expected number of riders per week



\---



**## 🛠️ Tech Stack**

\- Python

\- Streamlit

\- Scikit-learn

\- Pandas

\- NumPy

\- Pickle  



\---





**## ▶️ How to Run Locally**



**1. Clone the repository:**



git clone https://github.com/adarsh020106/ride-demand-prediction.git



**2. Navigate to the project folder:**



cd ride-demand-prediction



**3. Install dependencies:**



pip install -r requirements.txt



**4. Run the app:**



streamlit run app.py
